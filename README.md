# 概要
[florr.io JPwiki](https://newflorrio.wiki.fc2.com)で使用されるソースコード群。

# 構造
主に特殊構文のソースコード管理用です。
<br>
スタイルシートやフリーエリアのHTMLファイル等、第三者が編集可能なflorr.io JPwikiの一連の関連ファイルも含みます。
<br>
github pageを通じて、wikiのクライアント側からこれらのファイルを呼び出し実行しています。
